# Content Guide

Add content as Markdown files under `content/<module>/`.

## Front matter

Use this optional block at the top of each file:

```md
---
title: Page title
date: 2026-06-30
category: general
summary: Short summary
---
```

Supported fields:

- `title`
- `date`
- `category`
- `summary`

## Markdown support

The first version supports headings, paragraphs, and unordered lists.
